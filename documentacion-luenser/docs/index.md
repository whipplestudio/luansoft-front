# Documentación del Sistema de Gestión Fiscal Luenser

Bienvenido a la documentación oficial del sistema de gestión fiscal de Luenser. Este sitio contiene toda la información necesaria para utilizar eficientemente nuestro sistema, organizada por roles de usuario y módulos funcionales.

## Acerca del Sistema

El sistema de gestión fiscal de Luenser es una plataforma integral diseñada para facilitar la administración de procesos fiscales, contadores, clientes y documentos relacionados. Nuestro sistema permite:

- Gestión eficiente de contadores y sus clientes asignados
- Seguimiento de procesos fiscales con indicadores visuales
- Administración de documentos y entregables fiscales
- Monitoreo de estadísticas y métricas clave
- Control de acceso basado en roles

## Roles de Usuario

El sistema cuenta con diferentes roles de usuario, cada uno con permisos específicos:

- **Administrador**: Acceso completo a todas las funcionalidades del sistema
- **Contador**: Gestión de clientes asignados y sus procesos fiscales
- **Contacto**: Acceso limitado a información específica de clientes
- **Dashboard**: Visualización de estadísticas e indicadores clave

## Primeros Pasos

Para comenzar a utilizar el sistema:

1. Solicite sus credenciales de acceso al administrador del sistema
2. Ingrese al sistema utilizando la URL proporcionada
3. Introduzca su nombre de usuario y contraseña
4. Navegue por los módulos disponibles según su rol
